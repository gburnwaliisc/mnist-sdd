# ===========================================================================

# Python 3.10+  |  CPU and CUDA supported
# ===========================================================================
#
# Quick start:
#   pip install -r requirements.txt
#
# For GPU (CUDA) — replace torch + torchvision with the CUDA-enabled wheels
# from https://pytorch.org/get-started/locally/ before running the above.
# Example for CUDA 12.8:
#   pip install torch torchvision \
#       --index-url https://download.pytorch.org/whl/cu128
#
# ---------------------------------------------------------------------------
# torch / torchvision VERSION PAIRING (critical — see CLAUDE.md §11.2)
# ---------------------------------------------------------------------------
# Mismatched versions cause silent incorrect transform behaviour, not an
# import error.  The torchvision minor version must equal torch minor - 15:
#
#   torch 2.1  →  torchvision 0.16
#   torch 2.2  →  torchvision 0.17
#   torch 2.3  →  torchvision 0.18
#   torch 2.4  →  torchvision 0.19
#   torch 2.5  →  torchvision 0.20
#   torch 2.6  →  torchvision 0.21
#   torch 2.7  →  torchvision 0.22
#   torch 2.12 →  torchvision 0.27    ← verified working environment
#
# See https://github.com/pytorch/vision#installation for the full matrix.
# ---------------------------------------------------------------------------
#
# For an exact reproducible lock of the entire dependency tree, run:
#   pip freeze > requirements-lock.txt
# after installation.
# ===========================================================================

#   - torch.backends.cudnn deterministic flags (stable 2.x)

# Must form a matched pair with torch (see pairing table above).

# Tested: 0.27.0+cpu
